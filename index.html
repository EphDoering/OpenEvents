Open Events

A Google Calendar Bulk Editor

This tool lets you search for events you would like to edit, and then edit specfic feilds to assign to all of the found events. For example, one could search for "dentist" and set the color of all of those events to red.

No data is directly being stored locally or remotely by this tool. However, the google API is storing authentication cookies locally, saving the authorization. Additioanally, the google api is used to search and modify your calendars.
